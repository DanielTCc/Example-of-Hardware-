# Ferretería Mi Casa · Sistema POS e Inventario

Repositorio para mejorar el diseño de la ferretería **Mi Casa**. Este proyecto contiene una aplicación web (frontend) para gestión de **ventas (POS/Caja)** e **inventario**, construida principalmente con **TypeScript**.

## 🚀 Características
- Dashboard con gráficas
- Punto de venta (POS/Caja)
- Gestión de inventario
- UI demo con datos **mock** (sin base de datos)
- Moneda: **C$ (NIO)**

## 📦 Tecnologías
- TypeScript
- React
- Material UI (MUI)
- Vite

## ✅ Requisitos
- Node.js 18+

## ▶️ Ejecutar el proyecto
La app vive dentro de la carpeta `app/`:

```bash
cd app
npm install
npm run dev
```

## 📁 Estructura
- `app/` : Aplicación web (frontend)

## 📝 Notas
Este repositorio está enfocado en mejorar el diseño y experiencia de usuario.
